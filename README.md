## Sobre o projeto:

- Meses atrás eu havia feito a versão 1 com javascript, bootstrap, styled components e lottie. Recentemente vi o projeto e a necessidade de atualizá-lo com melhores práticas, refatoração e limpeza geral.

- Apesar de refatorado, não quis trocar o bootstrap pelo tailwind(lib que utilizo preferencialmente hoje em dia), mas a reclamação de CSS não utilizado do Core Web Vitals me incomodou. Com isso, implementei o PurgeCSS no PostCSS e passei a utilizar SASS nas importações. Problema resolvido.
